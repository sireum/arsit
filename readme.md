## Building Arsit's CLI

1. Build Arsit's Uber-Jar

    ```bash
    git clone --recursive git@github.com:santoslab/arsit.git
    ./arsit/bin/test.sh
    ```

2. Display Arsit's CLI options

    ```bash
    ./arsit/bin/arsit -h
    ```
